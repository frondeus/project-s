Before i continue i think it makes sense to introduce type ascription.


```
(: :number 5)
```

```json
5.0
```

```
(: :string "4")
```

```json
"4"
```

```
(: :string 5)
```

```json

   ╭─[ <input>:1:1 ]

   │       │    │   
   │       ╰──────── Expected string
   │            │

```

```
(: :bool true)
```

```type
Bool
```

## Type ascription in let

```
(let :foo (: :number 5))
foo
```

```type
Number
```

## Tuples

```
(: (tuple :number :string) (tuple 1 "2"))
```

```type
(Number, String)
```


## Functions

```
(: (fn (tuple :number :number) :number) 
   (fn (:a :b) (- a b))
)
```

```type
(Number, Number) -> Number
```
