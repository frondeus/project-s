I tak jak algebraicznie można powiedzieć że np dwa typy są identyczne strukturalnie
{ foo: number } & { bar : number } == { foo: number, bar: number }
Tak enumy również:
enum { None: () } & enum { Some: ('a) } == enum { None: () | Some: ('a) }
