Okay so, what i need is type constructor

It is basically a function that takes a type and returns another type.

It kinda lives both in the type system and in runtime.

```s
(type :Option 'a (enum
  :Some 'a
  :None ()
))

(let :o (: (Option :number) (enum :Some 1)))
```

```eval
val o : enum  {Some: (number) | None: ()} = {
  "Some": 1.0
}
- : () = []
```

Okay, that did not work, we still got `(1)`
The problem is, when we do `(Option :number)` we need to basically replace every occurrence of `'a` with `:number`.

Great!
