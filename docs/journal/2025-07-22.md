It's time to add type aliases


```s
(type :my-number :number)

(let :foo (: my-number 1))
```

```eval
val foo : number = 1.0
- : () = []
```

Okay, now i can use it with enums:

```s
(type :Option (enum
    :Some 'a
    :None ()
))

(let :foo (: Option (enum :Some 1))

```

```eval
val foo : enum  {Some: (1) | None: ()} = {
  "Some": 1.0
}
- : () = []
```
