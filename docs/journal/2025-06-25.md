## Implementing Super with Overlays

To support `super` like in Nix overlays, you need a `fix` function that can handle both self-reference and super (the previous/extended object). Here's how:

```
(let :fix (fn (:f) (do (let* :result (f result)) result )))

(let :fix-extend (fn (:ext :base) (do
    (let :base-inst (fix (fn (:self) (base self (obj/plain)))))
      
    (fix (fn (:self) (ext self base-inst)))
)))

(let :base-pkgs (fn (:self :super) 
    (obj/plain
        :a 1
        :b 2
        :c (thunk () (- (self :b) (self :a)))
    )
))

(let :extended-pkgs (fn (:self :super) 
    (obj/extend super
        :a 10
        :d (thunk () (- (self :a) (super :c)))
        :c 2
    )
))

(fix-extend extended-pkgs base-pkgs)
```

```type
{a: 10, b: 2, c: 2, d: number}
```

```json
{
  "a": 10.0,
  "b": 2.0,
  "c": 2.0,
  "d": 9.0
}
```

Okay, what about root?

```
(let :fix (fn (:f) (do
    (let* :result (f result))
    result
)))

(let :fix-root (fn (:root :f) (do
    (let* :result (f result root))
    result
)))

(fix (fn (:self) (obj/plain
    :a 1
    :b (thunk ()
        (fix-root self (fn (:self :root)
            (obj/plain
                :c (root :a)
            )
        ))
    )
) ))
```

```type
{a: 1, b: {c: 1}}
```

```json
{
  "a": 1.0,
  "b": {
    "c": 1.0
  }
}
```

Nice!
