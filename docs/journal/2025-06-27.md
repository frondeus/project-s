## Recursive types

```
(let* :f (fn () f))
f
```

```type
<() → 'a> as 'a
```

With type annotations:

```
(let* :f (: 
    (as 'a (fn () 'a))
    (fn () f)
))
f
```

```type
<() → 'b ∨ () → <() → 'a ∨ () → 'b> as 'a> as 'b
```
