## List find

Finds element based on predicate.
But first, I added Some/None and Option to prelude.

```s
(Some 1)
```

```eval
- : enum  {Some: 1} = {
  "Some": 1.0
}
```

```s
(let :l [1 2 3])
(list/find l (fn (:x) (= x 2)))
```

```eval
val l : [1 ∨ 2 ∨ 3] = [
  1.0,
  2.0,
  3.0
]
- : enum  {Some: 1 ∨ 2 ∨ 3 | None: ()} = {
  "Some": 2.0
}
```

Nice.
