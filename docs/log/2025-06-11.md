# Destructing

Next step is destructing.
Let's start with simplest example

```
(let (:a :b :c) (list 1 2 3))

(+ a b c)
```

```json
6.0
```

Destructing nested lists

```
(let (:a (:b :c) :d) (list 1 (list 2 3) 4))

(+ a b c d)
```

```json
10.0
```


Destructing objects

Simple name destructing 

```
(let { :a :b :c } { :a 1 :d 4 :b 2 :e 6 :c 3})

(+ a b c)
```

```json
6.0
```

Renaming

```
(let { :a b } { :a 1 :b 2 :c 3})

b
```

```json
1.0
```

Nested patterns

```
(let { :a { :b } :c } { :a { :b 1 } :c 3})

(+ b c)
```


```json
4.0
```

What about destructing in other places than let?
