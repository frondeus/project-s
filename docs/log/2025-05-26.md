Ok, now we should be able to clean the object adding.

```
(let :add-obj (macro (:key :value) 
  `(if (has? super ,key)
    '(,key (+ (super ,key) ,value))
    '(,key ,value)
  )
))
(let :left {
    (let :x 4)
    :another x
    :key {
      :a 1
      :b (+ 1 (root :another))
    }
  })

(let :right (thunk () {
  :another 9
  
  (add-obj :key {:c 3})
}))

(+ left right)
```

```json
{
  "another": 9.0,
  "key": {
    "a": 1.0,
    "b": 10.0,
    "c": 3.0
  }
}
```
